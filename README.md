# QuickNote
轻量简洁的浏览器侧边栏笔记扩展，随时记录、不打断浏览，简单好用。
## 功能特性
- 📝 极简笔记编辑，无多余操作
- 📦 浏览器原生侧边栏，不占用页面空间
- 💾 本地自动保存，数据不丢失
- 🔌 开箱即用，无需注册与配置
## 安装步骤
1. 克隆 / 下载本项目代码
2. 打开浏览器「扩展管理」（chrome://extensions/）
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择项目文件夹
## 使用方式
1. 点击扩展图标打开侧边栏笔记
2. 直接输入内容，自动实时保存
3. 关闭侧边栏后下次打开仍可恢复内容
## 文件结构
```plaintext
quicknote/
├── background.js    # 扩展后台逻辑
├── manifest.json     # 扩展配置清单
├── sidepanel.css     # 侧边栏样式
├── sidepanel.html    # 侧边栏界面
└── sidepanel.js      # 笔记交互逻辑
```
